# FrameTalk Web Mode
There is one part of FrameTalk system written in Web Mode: GameModule.
## Installation
- #### Required tools
  - install [python3](https://www.python.org/downloads/)
  
- #### Required python packages
  - for GameModule
    ``` 
    cd FrameTalk_Web
    pip3 install -r requirement.txt
    ```
    
## Settings
 - #### GameModule
    1. ```cd FrameTalk_Web```
    2. set everything in 'config.py'
    
## How to start
- #### GameModule
    ```
    cd FrameTalk_Web
    python3 server.py
    ```
